# Jay Chou NFT Ticket Coursework

This repository contains the Solidity smart contract for an NFT based concert ticketing system developed as part of coursework.

## Smart Contract Details
- Contract Name: JayChouConcertTicket
- Network: Sepolia Testnet
- Contract Address: 0xf263cc897B7A548b4392afe9933828a4773ff2A3

## Etherscan Link
https://sepolia.etherscan.io/address/0xf263cc897B7A548b4392afe9933828a4773ff2A3

## Description
The smart contract demonstrates a simplified NFT ticketing system where each token represents a concert ticket. The contract allows tickets to be minted and assigned to wallet addresses, simulating ownership and access rights.

A sample NFT ticket was successfully minted on the Sepolia test network to demonstrate functionality.
