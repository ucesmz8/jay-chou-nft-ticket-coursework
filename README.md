# Jay Chou NFT Ticket Coursework

This repository contains the Solidity smart contract for an NFT based concert ticketing system designed for coursework submission.

## Smart Contract
- Contract name: JayChouConcertTicket
- Network: Sepolia Testnet
- Smart contract address: 0xf263cc897B7A548b4392afe9933828a4773ff2A3

## Etherscan Link
https://sepolia.etherscan.io/address/0xf263cc897B7A548b4392afe9933828a4773ff2A3

## Description
The contract demonstrates a simple NFT style ticketing prototype where concert tickets can be minted and assigned to wallet addresses.
